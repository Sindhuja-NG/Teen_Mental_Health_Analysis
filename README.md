##Dataset Used:

Teen mental Health Dataset
   
   -The Dataset contains 1201 rows and 13 columns
   
   -There are no missing values in the Dataset
   
   -The Dataset is a structured type Dataset


##Key Obdervations

1.The dataset includes both Numerical(age,sleep hours,Screen time) and categorical(gender,Ssocial_interaction_level,Platform_usage)data

2.The Dataset appears with no duplicate entries or missiing values making the dataset clean and ready for analysis

3.The average screen time before sleep is 1.74hrs which may negatively impact sleep quality

4.Most students fall under moderate stress level like(4-7 range)rather than the extreme value 10.
